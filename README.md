# ZakyAI Ultimate - المساعد الذكي المتكامل
تطبيق Android + سيرفر Python + لوحة تحكم ويب مع دعم Groq, Cerebras, OpenRouter, Hugging Face.

## التشغيل السريع
1. Android: ضع google-services.json في android/app/، ثم اشغل التطبيق.
2. Server: cd server && pip install -r requirements.txt && python app.py
3. Web: افتح web/dashboard/index.html في المتصفح.
